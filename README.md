# MERN AI Flow Builder

## 🚀 Features
- React Flow based UI (Input → Result)
- AI Integration using OpenRouter
- MongoDB data storage
- Run & Save functionality

## 🛠 Tech Stack
- Frontend: React + React Flow
- Backend: Node.js + Express
- Database: MongoDB
- AI API: OpenRouter

## ▶️ How to Run

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm run dev

## 🔑 Environment Variables

Create `.env` in backend:

MONGO_URI=your_mongodb_url  
OPENROUTER_API_KEY=your_key  
PORT=5000  

## 📌 Notes
Free models on OpenRouter may not always be available, so fallback model is used.
